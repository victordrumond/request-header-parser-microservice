Request Header Parser Microservice

A freeCodeCamp project from the Back End Development and APIs Certification

Usage

A request to /api/whoami returns a JSON object containing the IP address, preferred languages and system infos for your device:

/api/whoami

Example Output

{"ipaddress": "::ffff:159.20.14.100", "language": "en-US,en;q=0.5", "software": "Mozilla/5.0 (X11; Ubuntu; Linux x86_64; rv:50.0) Gecko/20100101 Firefox/50.0"}
